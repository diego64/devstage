### ⚙️ Installation

<h4>From <code>source</code></h4>

> 1. Clone the repository:
>
> ```console
> $ git clone https://github.com/diego64/devstage/web
> ```
>
> 2. Change to the project directory:
> ```console
> $ cd devstage/web
> ```
>
> 3. Install the dependencies:
> ```console
> $ npm install
> ```
> 6. Start the application:
> ```console
> $ npm run dev
> ```
